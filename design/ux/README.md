## Scope

UX design assets, wireframes, and interaction specifications for all dashboard modules.

---

## Design Principles

1. **Executive-grade clarity** — information hierarchy designed for 5-second comprehension
2. **Arabic-first** — all wireframes designed in RTL by default
3. **Data density without clutter** — maximum information, minimum noise
4. **Accessible** — WCAG 2.1 AA minimum
5. **Print-ready** — key dashboards optimized for boardroom printing/projection
